# Arrays

Svar på oppgavene under hvert spørsmål i ```js/script.js```, du kan skrive kommentarer i filen men la svaret ditt stå uten kommentar. 
f.eks:

```js

// Make an array with 4 items...
var fruits = ["apple", "orange", "banana", "pear"]
```
